Channel Shift Venus Glitch

30. December 2020

Channel shifting is the process of taking the red, green or blue value of pixels in an image and applying those values to pixels in different channels. Here is an example of a pixels-based glitch effect in a p5.js sketch. It uses simple runs of pixel copying of variable length – sometimes with RGBA channel shifting.

As it is hard to put some definition around what "Glitch Art", I've managed to read by images in as byte arrays using loadBytes() method, then random delete bytes and reposition and resize the array so their aren't any gaps.
